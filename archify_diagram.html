Método Rota API Módulo / Serviço Descrição do Funcionamento
POST /api/import parser.ts Processa arquivos da LOA e salva registros de orçamento no banco.
POST /api/receitas-historicas parser.ts Realiza a ingestão de receitas com cálculo de hash SHA-256 para evitar duplicatas.
GET /api/loa dashboard-data.ts Retorna despesas filtradas por órgão, unidade, função, ação e exercício.
GET /api/loa/resumo presentation-data.ts Agrega os totais gerais para exibição no modo apresentação institucional.
